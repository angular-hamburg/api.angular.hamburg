
# api.angular.hamburg

**This is a prototype of a simple API for angular.hamburg.**

## API

So far, the API only has 1 method:

### `GET /next-meetup`

```json
{
  "date": "January 11, 2017"
}
```

## Development

- **`npm start`**: starts the API server on `localhost:80`
